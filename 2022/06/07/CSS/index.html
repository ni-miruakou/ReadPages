CSS cascading style sheets 层叠样式表

规则

具体性 specificity

级联性 cascade

继承

引入

外部

内部

内联，不推荐

结构

选择器selector

  • [label]

  • .[class]

  • [id]

  • :[pseudo-class] 伪类

  • ::[pseudo-elements] 伪元素

  • [[attr]=[value]]、[[attr]]

    ＝修饰

    • ~=包含独立的value
    • |=以value为前缀
    • ^=以value开头
    • $=以value结尾
    • *=包含一到多个独立非独立的value

    尾巴修饰

    • i 忽略大小写，支持ASCII外
    • s 区分大小写，支持ASCII外

多重选择

  • A,B 且
  • A+B A接下来的B
  • A~B A以后的B
  • A>B A下的B
  • A B A内的B

选择器的优先级

内联最高，ID，类/属性/伪类，元素/伪元素最低

属性

ALL

/ Global values /
all: initial; 初始值
all: inherit; 继承值
all: unset; 默认方式获得的值

/ CSS Cascading and Inheritance Level 4 /
all: revert; 回滚

速记

-top -right -bottom -left

-color -image -position -repeat -attachment
