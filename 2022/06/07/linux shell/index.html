linux shell

命令：[command] [-option] [-…] [arguments]
[command] [-option][…] [arguments]
tab键可以用于补全命令

特殊字符:::::
命令控制：
;分隔命令
\延续命令到下一行

通配符：
*任意字符或字符串
?任意单个字符
[字符组]字符组中任一字符
[-]

重定向：
shell打开标准输入(键盘)，标准输出(屏幕)，标准错误输出(屏幕)三个文件，用重定向符可以更改输入输出流
[命令]<[输入文件] 指定输入文件为命令的输入
[命令]<<[分隔符] [多行字符串] [分隔符]指定分隔符之间的部分为命令的输入
[命令]>[输出文件]指定命令的输出输出到输出文件，不存在则创建，存在则覆盖
[命令]>>[输出文件]指定命令的输出追加到输出文件
[命令]2>[输出文件]指定命令出错时输出到输出文件
[命令]2>>[输出文件]指定命令出错时输出追加到输出文件

管道：
[命令] | [命令]将上一个命令的输出输入到下一个命令

引用：
[字符] 忽略字符原有的命令含义
‘[字符串]’忽略字符串所有特殊字符的命令含义
“[字符串]”对 $ \ ` 保留命令含义，其他字符则忽略
::::::

命令列表：
ls 列出目录内容
-a显示隐藏目录和文件、
-d只显示目录
-R列出目录和子目录下的所有文件
-pF在文件名后添加说明文件类型的字符，/表示目录，@表示链接，|表示FIFO，=表示套接字
-Q输出文件名用双引号引用
-l显示详细信息
-o显示详细信息，排除组信息
-s显示文件大小
-k显示文件大小为多少k
-u按照上次访问顺序排序
-t按照时间排序
-S按照大小排序
-x按列输出，横向排序
-C按列输出，纵向排序
-r反向排序目录
-q以?代替不可输出的字符
-m横向输出文件名，以逗号分隔
pwd 显示当前工作目录
cd 改变当前工作目录
cd ~ 回到默认工作目录
cd ~[用户名] 回到用户名的默认工作目录
cd .. (cd../) 返回上级目录
cd / 返回到根目录

passwd 设置密码(termux-auth)
passwd [用户名]修改任意用户的密码(root)
su 成为超级用户(root)

- 加载root环境变量
-c 只使用超级用户身份执行一条命令
-m不改变环境变量的情况下成为超级用户

data 显示系统时间
-s设置系统时间(root)
clear 清屏
man [命令] 得到指定命令的帮助信息
